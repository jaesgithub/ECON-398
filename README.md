# ECON-398: Introduction to Causal Methods (Spring 2025)
This repository contains class materials for UBC Vancouver School of Economics ECON 398. UBC ECON 398, Introduction to Causal Methods (formerly Introduction to Applied Economics), focuses on causal inference, empirical analysis, applied economic research, and a course project which is writing a research paper.

## Course Description
Theory and practice of the analytical application of economics. Causality, causal modeling, empirical analysis, and the use of theoretical models in policy evaluation and other practical applications.

## Class Details
- Schedule: Tuesdays/Thursdays 2:00-3:20 PM, IRC Floor 1 Room 1
- Instructor: [Dr. Jonathan Graves](https://economics.ubc.ca/profile/jonathan-graves/)
  - [email](jonathan.graves@ubc.ca)

## Topics
- Randomized controlled trials (RCTs) as the gold standard for causality.
- Quasi-experimental designs: instrumental variables (IV), regression discontinuity (RDD), difference-in-differences (DiD).
- Matching methods and propensity score techniques.
- Basic regression analysis (OLS) with causal interpretation.
